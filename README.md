# 36Days36Algorithms
The goal is to release a novel and unpublished encryption algorithm each day for the next 36 days.

Decryption algorithms will generally be left as an exercise to the reader.

Per [15 C.F.R. ยง742.15(b)(2)](https://www.law.cornell.edu/cfr/text/15/742.15), algorithm's URL and information will be forwarded to crypt@bis.doc.gov and to enc@nsa.gov.
